> [!IMPORTANT]
> Azure AI Inference beta SDK is deprecated and will be **retired on May 30, 2026**. If you're currently using this SDK with Microsoft Foundry Models or Azure OpenAI service, switch to the generally available [OpenAI/v1 API](https://aka.ms/openai/v1), which uses a stable OpenAI SDK. Follow the migration guide: [Migrate from Azure AI Inference SDK to OpenAI SDK](../how-to/model-inference-to-openai-migration.md) to switch to OpenAI/v1, using the SDK for your preferred programming language.
