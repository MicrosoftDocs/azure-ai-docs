> [!NOTE]
> Azure Data Lake Storage Gen1 was retired on *February 29, 2024*. If you use Azure Data Lake Storage Gen1, migrate to Azure Data Lake Storage Gen2. To learn how, see [Migrate Azure Data Lake Storage from Gen1 to Gen2 by using the Azure portal](/azure/storage/blobs/data-lake-storage-migrate-gen1-to-gen2-azure-portal).
>
> You can't create new Azure Data Lake Storage Gen1 accounts.
