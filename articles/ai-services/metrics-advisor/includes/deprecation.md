> [!IMPORTANT]
> Azure AI Metrics Advisor has been retired as of 18 May 2026.
>
> We recommend the following alternatives:
> 1. [Azure Monitor](/azure/azure-monitor/fundamentals/overview), as an official Azure 3P product, anomaly detection and analysis capabilities are provided through multiple interfaces, e.g., [Detect and analyze anomalies with KQL in Azure Monitor](/azure/azure-monitor/logs/kql-machine-learning-azure-monitor).
> 2. The [Open-Source Anomaly Detector](https://github.com/microsoft/anomaly-detector), this open-source project provides the same anomaly detection capabilities used by Kensho, Azure Metrics Advisor, and Azure Anomaly Detector in the backend.
> 3. Fabric
>    - [Overview & tutorial: Multivariate anomaly detection overview - Microsoft Fabric](/fabric/real-time-intelligence/multivariate-anomaly-overview)
>    - [Techcommunity blog: Advanced Time Series Anomaly Detector in Fabric](https://techcommunity.microsoft.com/blog/azuredataexplorer/advanced-time-series-anomaly-detector-in-fabric/4226195)
